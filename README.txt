VehicleInfoSystem
------------------------------------

INTRODUCTION
------------------------------------
A simple vehicle information database created with Java.


COMPILING THIS PROGRAM.
-------------------------------------
To compile this program run “javac -d class src/edu/uga/cs1302/vehicles/*.java” while in the Project 2 folder.



RUNNING THE PROGRAM
-------------------------------------
To execute this program you need to run the VehicleInfoSystem file using "java -classpath class edu.uga.cs1302.vehicles.VehicleInfoSystem"
